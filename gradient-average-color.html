The following repeating gradients should render as a solid colour equivalent to rgba(25%,25%,25%,1) () in cases where they cannot “faithfully render” the repeating pattern. See http://www.w3.org/TR/css3-images/#repeating-gradients for details.

What constitutes faithful rendering of a repeating gradient is not defined, but at a minimum, the first gradient in each row (d01, d06) should yield the solid colour shown previously, and almost certainly the second two in each row (d02, d07) should do the same.
