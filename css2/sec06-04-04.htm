CSS2 Test Suite: 6.4.4 Precedence of non-CSS hints

[index page] [section] [Previous] [Next] [Specification]

The style declarations contained within this page:

.cl1 {colour: green;}

This paragraph contains text which is contained by [cl1] a FONT element with a colour value of "red". This should be overridden by the CSS rule for class 'cl1', which takes precedence over the non-CSS presentational hint embodied by the FONT element. Therefore, the FONT-enclosed text should be green, not red.
