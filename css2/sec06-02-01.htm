The style declarations contained within this page:

P.cl1 {colour: red;}
A:link {colour: inherit; text-decoration: underline; font-weight: bold;}

[cl1] Although this paragraph has a class of 'cl1', it also has an inline STYLE of 'colour: inherit', which should cause it to inherit its colour from the BODY element. Thus, the colour should be black.

This unclassed paragraph contains a number of anchors which have been underlined and boldfaced, but whose colour inherits from the paragraph itself (thus making them black).
