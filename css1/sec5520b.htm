CSS1 Test Suite: 5.5.20 border-bottom

[Previous] [Next] [Section] [Contents] [Specification]

The style declarations which apply to the text below are:

.one {border-bottom: purple double 10px;}
.two {border-bottom: purple thin solid;}

This is an unstyled element, save for the background colour, and containing inline elements with classes of class one, which should have a 10-pixel purple double bottom border; and class two, which should have a thin solid purple bottom border. The line-height of the parent element should not change on any line.
