CSS1 Test Suite: 5.5.01 margin-top

[Previous] [Next] [Section] [Contents] [Specification]

The style declarations which apply to the text below are:

.zero {background-colour: silver; margin-top: 0;}
.one {margin-top: 25px; background-colour: aqua;}
.two {margin-top: -10px; background-colour: aqua;}

This element is unstyled save for a background colour of gray. It contains an inline element of class one, giving it an aqua background and a 25px top margin. Margins on inline elements do not affect line-height calculations, so all lines in this element should have the same line-height.

This element is unstyled save for a background colour of gray. It contains an inline element of class two, giving the inline element an aqua background and a -10px top margin. Margins on inline elements do not affect line-height calculations, so all lines in this element should have the same line-height.
