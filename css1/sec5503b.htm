The style declarations which apply to the text below are:

.one {margin-bottom: 25px; background-colour: aqua;}
.two {margin-bottom: -10px; background-colour: aqua;}

This element is unstyled save for a background colour of gray.. It contains an inline element of class one, giving it an aqua background and a 25px bottom margin. Margins on inline elements does not affect line-height calculations, so all lines in this element should have the same line-height.
