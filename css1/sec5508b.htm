CSS1 Test Suite: 5.5.08 padding-bottom

[Previous] [Next] [Section] [Contents] [Specification]

The style declarations which apply to the text below are:

.one {padding-bottom: 25px; background-colour: aqua;}
.two {padding-bottom: -10px; background-colour: aqua;}

This element is unstyled save for a background colour of gray. It contains an inline element of class one, giving it an aqua background and a 25px bottom padding. Padding on inline elements does not affect line-height calculations, so all lines in this element should have the same line-height. There may be implementation-specific limits on how much of the padding the user agent is able to display.

This element is unstyled save for a background colour of gray. It contains an inline element of class two, giving it an aqua background and no bottom padding, since negative padding values are not allowed. Padding on inline elements does not affect line-height calculations, so all lines in this element should have the same line-height.
