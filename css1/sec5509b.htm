CSS1 Test Suite: 5.5.09 padding-left

[Previous] [Next] [Section] [Contents] [Specification]

The style declarations which apply to the text below are:

.one {padding-left: 25px; background-colour: aqua;}
.two {padding-left: -10px; background-colour: aqua;}

This element is unstyled save for a background colour of gray. However, it contains an inline element of class one which should result in 25-pixel left padding (which should also be a light blue) only in the first line in which the inline box appears.

This element is unstyled save for a background colour of gray. However, it contains an inline element of class two, giving it an aqua background and no bottom padding, since negative padding values are not allowed. Padding on inline elements does not affect line-height calculations, so all lines in this element should have the same line-height.
