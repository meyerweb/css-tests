CSS1 Test Suite: 5.5.07 padding-right

[Previous] [Next] [Section] [Contents] [Specification]

The style declarations which apply to the text below are:

.one {padding-right: 25px; text-align: right; background-colour: aqua;}
.two {padding-right: -10px; text-align: right; background-colour: aqua;}

This element is unstyled save for a background colour of gray. However, it contains an inline element of class one, which should result in 25-pixel right padding (which should also be light blue) only in the last line in which the inline box appears.

This element is unstyled save for a background colour of gray. However, it contains an inline element of class two, which should result in no right padding, since negative padding values are not allowed, in the last line in which the inline box appears.
